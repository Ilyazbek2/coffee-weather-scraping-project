# Coffee, Weather & Web Scraping Mini Project

This project contains three Jupyter notebooks:

1. `coffee_analysis.ipynb`  
   - Coffee sales analysis:
     - Sales by day of week (bar chart)
     - Price distribution by coffee type (boxplot)
     - Weekend vs weekday sales (bar chart)

2. `books_scraping.ipynb`  
   - Scraping book titles from http://books.toscrape.com using `requests` + `BeautifulSoup`.

3. `weather_api.ipynb`  
   - Getting current weather for a city using OpenWeatherMap API.

## How to run

1. Create and activate a virtual environment (optional).
2. Install dependencies:
   ```bash
